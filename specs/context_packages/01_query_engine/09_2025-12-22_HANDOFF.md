## Project Goal

Build a **two-layer intelligence system** for understanding Claude Code usage patterns:

1. **Deterministic Index Layer** (THIS WORK) - Fast primitives from parsed JSONL data, NO LLM
2. **Intelligent Agent Layer** (Future) - Reasoning and judgment on top of indexes

**Key Insight:** Claude Code's `leafUuid` mechanism enables chain detection - when a session continues, it stores `leafUuid` pointing to parent session's last message UUID.

## Phase 5 Just Completed

### Bloom Filters (`bloom.py`)
Pure Python implementation (no deps) for O(1) probabilistic membership:

```python
from context_os_events.index import BloomFilter

bloom = BloomFilter(expected_items=1000, false_positive_rate=0.01)
bloom.add("/src/main.py")

if "/src/main.py" in bloom:
    print("Probably in set")  # May have false positives

if "/other.py" not in bloom:
    print("Definitely NOT in set")  # Never false negatives
```

### Temporal Buckets (`temporal.py`)

from context_os_events.index import (

## Phase 6: Unified ContextIndex (NEXT)

### Purpose
Single interface wrapping ALL index structures for clean agent queries.

### Class Design

```python
from pathlib import Path
from typing import Dict, List, Optional, Tuple
from datetime import datetime

class ContextIndex:
    """Unified interface to all index structures."""

    def __init__(self, jsonl_dir: Optional[Path] = None):
        """Initialize with in-memory indexes or from database."""
        self._chains: Dict[str, Chain] = {}
        self._inverted_index: Dict[str, List[FileAccess]] = {}
        self._file_tree: Optional[FileTreeNode] = None
        self._co_access: Dict[str, List[Tuple[str, float]]] = {}
        self._temporal: Dict[str, TemporalBucket] = {}

    # === Chain Queries ===
    def get_chain(self, chain_id: str) -> Optional[Chain]:
        """Get chain by ID."""

    def get_chain_for_session(self, session_id: str) -> Optional[str]:
        """Get chain ID containing session."""

    def get_all_chains(self) -> List[Chain]:
        """Get all chains sorted by recency."""

    # === File Queries ===
    def get_sessions_for_file(self, file_path: str) -> List[FileAccess]:
        """Get all sessions that touched a file."""

    def get_files_for_session(self, session_id: str) -> List[str]:
        """Get all files touched by a session."""

    def get_co_accessed(self, file_path: str, limit: int = 5) -> List[Tuple[str, float]]:
        """Get files frequently accessed with this one (game trails)."""

    # === Directory Queries ===
    def get_directory_stats(self, dir_path: str) -> Dict:
        """Get session/chain counts for a directory."""

    def get_hot_directories(self, limit: int = 10) -> List[Tuple[str, int]]:
        """Get directories with most session activity."""

    # === Temporal Queries ===
    def get_week_summary(self, week: str) -> Optional[TemporalBucket]:
        """Get summary for ISO week (e.g., '2025-W50')."""

    def get_recent_weeks(self, count: int = 4) -> List[TemporalBucket]:
        """Get N most recent weeks with activity."""

    def get_weeks_in_range(self, start: datetime, end: datetime) -> List[TemporalBucket]:
        """Get all weeks overlapping date range."""

    # === Fast O(1) Bloom Checks ===
    def chain_touched_file(self, chain_id: str, file_path: str) -> bool:
        """O(1) check if chain probably touched file."""

    def file_touched_in_week(self, file_path: str, week: str) -> bool:
        """O(1) check if file was probably touched in week."""

    # === Build Methods ===
    @classmethod
    def build_from_jsonl(cls, jsonl_dir: Path) -> "ContextIndex":
        """Build all indexes from JSONL directory."""
        instance = cls()
        instance._chains = build_chain_graph(jsonl_dir)
        instance._inverted_index = build_inverted_index(jsonl_dir, instance._chains)
        instance._file_tree = build_file_tree(instance._inverted_index)
        instance._co_access = build_co_access_matrix(instance._inverted_index)
        instance._temporal = build_temporal_buckets(instance._inverted_index)
        return instance

    # === Persistence ===
    def persist(self, db_path: Path) -> Dict[str, int]:
        """Persist all indexes to SQLite database."""

    @classmethod
    def load(cls, db_path: Path) -> "ContextIndex":
        """Load all indexes from SQLite database."""
```

## Key Implementation Notes

1. **No LLM in Index Layer** - Pure computation from parsed data
2. **Bloom Filter Params** - `expected_items=1000, false_positive_rate=0.01`
3. **Windows Paths** - Normalized to forward slashes in file_tree
4. **ISO Weeks** - Format `YYYY-WXX` (e.g., "2025-W50")
5. **Session ID = Filename** - JSONL files named by UUID
